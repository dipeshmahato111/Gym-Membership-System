##  Project Overview
This is a Java-based application designed to manage the core functionalities of a gym membership system. The project demonstrates advanced Object-Oriented Programming (OOP) concepts such as **Inheritance**, **Encapsulation**, and **Method Overriding**. 

The system allows for multi-user authentication and differentiates between standard members and administrative oversight.

##  System Features
- **User Authentication:** Secure login/logout logic for all users.
- **Member Management:** Ability to view specific membership plans and status.
- **Admin Controls:** Functionality for administrators to manage and monitor member accounts.
- **Membership Lifecycle:** Logic for activating and deactivating memberships.

##  Project Structure
The repository contains the following core Java classes:
- `User.java`: The base class containing common attributes (ID, Name, Email).
- `GymMember.java`: Subclass of User for member-specific features.
- `GymAdmin.java`: Subclass of User for administrative functions.
- `Membership.java`: Handles subscription details and status logic.
- `Main.java`: The test suite that demonstrates the system with multiple actors (Dipesh and Raj).
